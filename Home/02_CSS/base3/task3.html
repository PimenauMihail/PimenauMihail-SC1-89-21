Блок навигации

  • Ширина 15em
  • Должен присутсвовать цвет фона
  • Располагается слева
  • Блок навигации наполните контентом достаточным,чтобы по высоте он захватывал вторую статью, как показано в данном примере

Статья 1

Размер блока:
Занимает всю доступную ширину
Должен присутствовать цвет фона
Во всех "статьях" сделайте отступ снизу от соседей равным .2em, при помощи
свойства margin-bottom

Статья 2

Размер блока:

Должен присутствовать цвет фона

свойства margin-bottom Картинка вставленна при помощи тега, плаванье
организованно по правому краю внутри данной статьи для этой картинки
