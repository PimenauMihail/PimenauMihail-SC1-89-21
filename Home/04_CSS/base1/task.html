Форматы файлов

.DOC или .doc (от англ. (Document) - расширение имени файла, используемое для файлов,
представляющих текст, с разметкой или без. Расширение .DOC часто использовалось для обозначения
простых тестовых файлов без форматирования, однако позже стало использоваться для двоичных
форматов с разметкой.
(Документ МС WORD)

Portable Document Format (PDF) - межплатформенный формат электронных документов,
разработанный фирмой Adobe Systems с использованием ряда возможностей языка PostSkript.
(Документ PDF)

ZIP - популярный форматсжатия данных и архивации файлов. Файл в этом формате обычно имеет
расширение (zip) и хранит в сжатом или нее сжатом виде один или нескоько файлов, которые можно
из него извлечь путем распаковки с помощью специальной программы.
(Архив)
